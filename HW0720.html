2. CSS가 포함된 웹 페이지를 작성하고 강의 자료에서 기술한 텍스트 글꼴 CSS 프로퍼티를 최소 한 번씩 사용하라.(GitHub에 등록 후 주소만 공유)

web-font

웹 폰트는 사용자에게 없어도, 웹에서 글꼴을 불러와서 사용한다. 한글 글꼴은 용량이 커서 적합하지 않습니다.

유명한 폰트라길래 한번 써봤습니다. abrilFatface입니다. 출처는.. 여기에요.

/* latin */ @font-face { font-family: 'Abril Fatface'; font-style: normal; font-weight: 400; src: local('Abril Fatface'), local('AbrilFatface-Regular'), url(http://fonts.gstatic.com/s/abrilfatface/v8/X1g_KwGeBV3ajZIXQ9VnDgYWpCd0FFfjqwFBDnEN0bM.woff2) format('woff2'); unicode-range: U+0000-00FF, U+0131, U+0152-0153, U+02C6, U+02DA, U+02DC, U+2000-206F, U+2074, U+20AC, U+2212, U+2215, U+E0FF, U+EFFD, U+F000; }

main #web-font{ font-family:'맑은고딕' 'NanumGothic', 'MalgunGothic'; font-size: 9pt; }

font-family, font-size

적용할 글꼴을 여러 개 적는다. 앞쪽에 쓰인 글꼴이 사용자가 지원하지 않는 글꼴일 가능성을 염두에 두고, 제일 마지막에는 기본글꼴중에 하나를 적는 것이 좋겠죠? 이 페이지의 소스코드 부분에 적용해 보았습니다.

font-family : 돋움, dotum, Monaco; font-size : 9pt;

color, font-style, text-align, font-weight

이 페이지의 네비게이션 바, 1번 문제 소스의 스타일 부분에 사용해 보았습니다.

header nav #menu_list li{ margin: 0px 0px 1px -1px; width : 140px; float:left; list-style: none; text-align: center; border-left:1px solid #AAAAAA; border-right: 1px solid #AAAAAA; } header #menu #menu_list li > a{ text-decoration: none; font-weight:bold; font-size:0.8em; } header #menu #menu_list li > a:visited { color:#AABBBB; } header #menu #menu_list li > a:focus { color:blue; } main #priority_score .sub_article #first_solution p span{ font-style: italic; }

vertical-align, text-decoration, text-transform, letter-spacing, line-height, word-spacing

이 페이지 상단의 헤더 제목에 사용하였습니다.

header #welcome{ word-spacing : 5px; letter-spacing: 0px; font-size: 1.0em; padding-left: 10px; line-height: 1.1em; } header #welcome span{ letter-spacing: 0px; vertical-align: text-top; font-size :0.8em; } header #welcome #my_name{ text-transform: uppercase; } header #welcome #korean_name{ text-decoration: line-through; }

text-indent, white-space􏰀, text-overflow

이 페이지의 설명부분에 text-indent를, 소스 부분에 white-spce:pre , text-overflow:ellipsis를 사용했습니다. 메뉴 네비게이션에 hover 슈도클래스에 text-shadow를 사용했습니다.

main .css_source p{ white-space:pre; overflow: hidden; text-overflow:ellipsis; } main #font_style p{ padding-left: 15px; text-indent: -5px; } #menu #menu_list li:hover{ text-shadow:1px 1px #778; background: #CCC; }
