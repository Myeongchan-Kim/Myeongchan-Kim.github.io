과제2 : 과거 XHTML 규격에서는 <script > 태그를 사용할 때 다음과 같이 CDATA 섹션을 두기도 했다.

과거 XHTML 규격에서는 <script > 태그를 사용할 때 다음과 같이 CDATA 섹션을 두기도 했다. 

<script type="text/javascript">
//<![CDATA[
...code...
//]]>
</script>
XHTML에서 이렇게 작성해야 했던 이유를 설명하라.

이는 script 안에 들어있는 특수문자를 일부 웹 브라우저가, HTML태그로 인식하는것을 막기 위해서 제공되었다. 만약 브라우저가 위 부분을 XHTML로 인식하면 CDATA 문법을 이용해서 그 안에 있는 특수문자를 HTML entity로 변환하고 스크립트 전체를 스크립트로써 이해하지만, 만일 HTML 태그로 인식한다면 오류가 발생하게 된다. 또한, CDATA[] 이 문법 자체를 자바스크립트는 해석하지 못하고 오류를 발생히키기 때문에 자바스크립트의 주석문구인 // 를 이용해서 선언부를 자바스크립트가 해석하지 못하도록 처리했다.
